== Installation ==

1. Install it to from your admin -panel
Устанавливаем из админки
2. Activate the plugin through the 'Plugins' menu in WordPress
Активируем
3. Place `<? my_domain_search() ?> in your templates. Or place WPHOIS widget on sidebar. Don`t use both methods at the same time.  
Вызываем виджет из сайдбара, или размещаем <? my_domain_search() ?> в шаблоне. Одновременно оба метода не реккомендую использовать

4. Default language is russian- to set english just rename english.php into default.php
